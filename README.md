# AuthCAS (module for Omeka-S)

## Requirements

- Omeka-S 3 or higher

## Install

    cd omeka-s-path/modules

    git clone git@github.ugent.be:GhentCDH/omekas_cas.git Cas
    cd AuthCAS
    composer install --no-dev
